# Handoff

This document is used when transferring responsibility for the project, a feature, or a set of tasks from one agent (human or AI) to another.

## Purpose

- Ensure smooth transitions
- Document current status, blockers, and next steps
- Provide context for the new agent or contributor

## Template

---
**Date:** [YYYY-MM-DD]
**From:** [Current agent/contributor]
**To:** [New agent/contributor]
**Scope of Handoff:** [What is being handed off]
**Current Status:** [Summary of progress]
**Outstanding Tasks:**

- [ ] Task 1
- [ ] Task 2
- ...
**Known Issues/Blockers:** [List any]
**Next Steps:** [Recommended actions]

---

(Complete and update this document for each handoff event.)
